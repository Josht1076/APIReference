Description

Collection that provides access to all of the existing mesh combine features in a component and supports the ability to create new mesh Combine features.

Methods

Name Description
add Creates a mesh combine feature.
classType Static function that all classes support that returns the type of the class as a string. The returned string matches the string returned by the objectType property. For example if you have a reference to an object and you want to check if it's a SketchLine you can use myObject.objectType == fusion.SketchLine.classType().
createInput Creates a MeshCombineFeatureInput object. Use properties and methods on this object to define the mesh combine you want to create and then use the add method, passing in the MeshCombineFeatureInput object.
item Function that returns the specified mesh combine feature using an index into the collection.
itemByName Function that returns the specified MeshCombine feature using the name of the feature.
