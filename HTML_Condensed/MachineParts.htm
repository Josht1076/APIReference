MachineParts Object

Derived from: Base Object
Defined in namespace "adsk::cam" and the header file is <Cam/Machine/MachineParts.h>

Description

Object that represents a collection of machine parts. These parts are the children of another part or the collection of base parts from MachineKinematics.

Methods

Name Description
add Add a new part to this collection. The part's parent will be set to the owner of this collection, or null if this is the root parts collection.

If the passed MachinePartInput has a MachineAxisInput a new MachineAxis will be created.
classType Static function that all classes support that returns the type of the class as a string. The returned string matches the string returned by the objectType property. For example if you have a reference to an object and you want to check if it's a SketchLine you can use myObject.objectType == fusion.SketchLine.classType().
createPartInput Create a new MachinePartInput.
item Get the part at index in this collection.
itemById Get the part with the given ID.
