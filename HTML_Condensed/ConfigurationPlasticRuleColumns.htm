ConfigurationPlasticRuleColumns Object

Derived from: Base Object
Defined in namespace "adsk::fusion" and the header file is <Fusion/Configurations/ConfigurationPlasticRuleColumns.h>

Description

Provides access to the columns in a plastic rule table. This collection can be empty when no columns have been created. When the table is empty, it is not displayed in the user interface, and adding a column causes the table to be displayed.

Methods

Name Description
add Adds a new column to the plastic rule table.
classType Static function that all classes support that returns the type of the class as a string. The returned string matches the string returned by the objectType property. For example if you have a reference to an object and you want to check if it's a SketchLine you can use myObject.objectType == fusion.SketchLine.classType().
item A method that returns the specified column using an index into the collection. These are returned in the same order as they appear in the table.
itemById A method that returns the column with the specified ID.
