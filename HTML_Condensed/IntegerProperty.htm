IntegerProperty Object

Derived from: Property Object
Defined in namespace "adsk::core" and the header file is <Core/Application/IntegerProperty.h>

Description

An integer value property.

Methods

Name Description
classType Static function that all classes support that returns the type of the class as a string. The returned string matches the string returned by the objectType property. For example if you have a reference to an object and you want to check if it's a SketchLine you can use myObject.objectType == fusion.SketchLine.classType().
getLimits Method that returns any limits for the value of this property. The HasLimits property can be used to see if there are any limits or not.

This is most commonly used for properties associated with materials and appearances.
