LinearMarkingMenu Object

Derived from: Base Object
Defined in namespace "adsk::core" and the header file is <Core/UserInterface/LinearMarkingMenu.h>

Description

Represents the linear marking menu which is the vertical menu that's displayed when the user right-clicks within Fusion. This supports customizing the contents of the context menu.

Methods

Name Description
classType Static function that all classes support that returns the type of the class as a string. The returned string matches the string returned by the objectType property. For example if you have a reference to an object and you want to check if it's a SketchLine you can use myObject.objectType == fusion.SketchLine.classType().
clear Completely clears the contents of the context menu. If left in this state, the context menu will not be displayed.

Properties

Name Description
controls Return the collection of top-level controls in the context menu. It's possible to have drop-down controls (fly-outs) that provide access to additional controls. You can remove and add controls to customize the contents of the context menu.
