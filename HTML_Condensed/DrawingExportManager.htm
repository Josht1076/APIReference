DrawingExportManager Object

Derived from: Base Object
Defined in namespace "adsk::drawing" and the header file is <Drawing/Drawing/DrawingExportManager.h>

Description

Provides support to export the drawing in various formats.

Methods

Name Description
classType Static function that all classes support that returns the type of the class as a string. The returned string matches the string returned by the objectType property. For example if you have a reference to an object and you want to check if it's a SketchLine you can use myObject.objectType == fusion.SketchLine.classType().
createPDFExportOptions Defines the various settings for a STEP export.
execute Executes the export operation to create the file in the format specified by the input ExportOptions object.
