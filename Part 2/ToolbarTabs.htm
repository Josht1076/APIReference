ToolbarTabs Object

Derived from: Base Object
Defined in namespace "adsk::core" and the header file is <Core/UserInterface/ToolbarTabs.h>

Description

Provides access to a set of toolbar tabs.

Methods

Name Description
add Creates a new ToolbarTab. The tab is initially empty. This method appends the tab to the end of the collection.
classType Static function that all classes support that returns the type of the class as a string. The returned string matches the string returned by the objectType property. For example if you have a reference to an object and you want to check if it's a SketchLine you can use myObject.objectType == fusion.SketchLine.classType().
item Returns the specified toolbar tab using an index into the collection. When iterating by index, the tabs are returned in the same order as they are shown in the user interface.
itemById Returns the ToolbarTab at the specified ID.
