Description

The VolumetricSampler object which is used for controled sampling of the volumetric model.

Methods

Name Description
addSamplePoints Appends sample points to the existing sample points to be used for sampling the volumetric model.
classType Static function that all classes support that returns the type of the class as a string. The returned string matches the string returned by the objectType property. For example if you have a reference to an object and you want to check if it's a SketchLine you can use myObject.objectType == fusion.SketchLine.classType().
clearSamplePoints Clears the sample points that have been set for sampling the volumetric model.
evaluate Evaluates the volumetric model at the previously set sample points for the given node and returns the results.
evaluateColor Evaluates the color of the model at the previously set sample points and returns the results.
evaluateDensity Evaluates the density of the model at the previously set sample points and returns the results. This value is what is used to determine the level set of the model.
evaluateLevelSet Evaluates the level set function of the model at the previously set sample points and returns the results.
samplePointCount Gets the number of sample points that will be used for sampling the volumetric model.
setBoundingBoxSampling Calculates and sets the sample points to be used for sampling the volumetric model for a given resolution throughout the bounding box provided. This will override any previously set sample points.
setPlaneSampling Calculates and sets the sample points to be used for sampling the volumetric model for a given resolution, plane and primary axis. The points will be distributed in a grid pattern on the plane, starting at the plane origin and extend in the primary axis and secodary axis for the axis size arguments. The secondary axis is calculated from the cross product of the plane normal and the primary axis. This will override any previously set sample points.
setSamplePoints Sets sample points to be used for sampling the volumetric model.
